# UX 시나리오 — Conflict View (멋진 신세계)

작성 시점: 2026-07-14  
기준: 지정학(반서방 4허브 nav) + 지경학 분리, ModePicker 세부창 없음.

제품 한 줄: **지정학 = 전선·축·OSINT / 지경학 = 에너지·물류·시장.**  
입구 브랜드 감각은 「멋진 신세계」, 지정학 *안쪽* 렌즈는 반서방 4허브.

## 1. 첫 방문 진입 (해피패스)

### 1-A. 로딩
1. 사이트 접속.
2. 지구본(또는 로딩 셰이더)이 **고도 ≈ 2.85** (광역)로 보임.
3. 로컬/클라우드 데이터 동기화 상태 표시 가능.

### 1-B. 주의 / 환영
1. (설정에 따라) 주의 게이트 → **환영 편지지** (`WelcomeParchmentLetter`).
2. 주의 창 **우상단 「스킵하시겠습니까?」** → 편지 생략, **지정학·지경학** 선택으로 직행.
3. 「확인했습니다」류 CTA → 환영 편지 → 도메인.
4. 도메인: 창 선택 시 **추가 세부 설정창 없이** 바로 히어로. **초기화 모드(가볍게)** 토글로 레이어를 줄인 빠른 입장.
   - 토글 상태는 즉시 저장되며, 이후 접속·새로고침·모드 전환에도 계속 적용된다.

### 1-C. 도메인 선택 (핵심 분기)
1. **지정학** vs **지경학(경제·시장)** 선택.
2. 선택 순간:
   - 뷰어 모드·크롬·기본 레이어 패키지 적용.
   - ModePicker 세부 전장 창 **없음** → 바로 히어로 지구본.

### 1-D. 지정학 히어로 (공통 첫 화면)
- 카메라: 광역(엔트리 고도와 동일 계열).
- 기본 ON (대략):
  - **전쟁구역(빨강 빗금) — 지정학 기본 ON, 주요전장 이동 후에도 유지**
  - 외교적 긴장구역(주황 빗금)
  - GDELT 전쟁 뉴스 핀
- 기본 OFF: ADS-B, AIS, 축 관계망 상시, ADIZ 등 (캡·렉 가드).
- **이 단계까지는 전원 공통.** 허브 nav·양피지로 자동 진입하지 않음.
- 상단: 검색 + **중국 · 러시아 · 북한 · 이란** 드롭다운(아직 설명 표지만 있을 수 있음).
- 우측 위 **「주요전장」** ExplorationTabs(대만·한반도·우크라이나·중동) — 버튼 클릭 시 해당 충돌지로 fly.

### 1-D2. 크롬 표지 온보딩 (도메인 직후 · 1회)
1. **상단 nav**를 가리키는 표지 + 설명창  
   → “직접 열어 보세요. 로딩/자동으로 들어가지 않습니다.”
2. **하단 뉴스·속보 창**을 가리키는 표지 + 설명창  
   → “지구본 아래 뉴스. 궁금할 때 탭하세요.”
3. 확인이 끝나야 자유 탐색(QuickStart 등)으로 이어짐.
4. 허브 메뉴·양피지·SIPRI/충돌사 패널은 **유저가 nav를 발견해 클릭한 뒤에만** 동작.  
   우측 「주요전장」드롭다운은 언제든 열어 충돌지로 이동할 수 있다.

### 1-E. 지경학 히어로
- 항로·초크포인트·항구 등 물류 골격 ON.
- GDELT/텔레그램 전선 페치 성격 약화/OFF.
- 상단: 검색 → 포커스 시 에너지·초크·금융 허브 메뉴.
- 하단: 증시·VIX·유가 티커 + 경제 RSS.
- nav 헤더 감각: 「멋진 신세계 · 시장」.
- 지정학과 동일하게 **nav / 하단 뉴스 표지**를 거친 뒤 자유 탐색.

### 1-F. 재방문
- 환영 게이트 스킵 가능(localStorage).
- 프로덕션에서만 레이어 prefs 유지(개발은 기본값 리셋 경향).

## 2. 지정학 — 상단 허브 Nav

> **전제:** §1 공통 진입·표지가 끝난 뒤, **유저가 상단 nav에 관심을 갖고 직접 클릭할 때만** 아래 경로가 열린다.  
> 로딩·환영·도메인 선택 중에는 허브 fly·양피지로 넘어가지 않는다.

각 허브 드롭다운 공통 구조:

1. **{국가} · 국경 · 우군 관계망**
2. **우군 국가** (하위 목록)
3. **영유권 주장 및 영향** (하위 목록)
4. **무기거래 (SIPRI)**
5. **반서방국 충돌사**

### 공통 UX (설명 우선)

1. 클릭 → **지도 soft fly** (bbox·강조·호·링을 잠깐 보여 줌).
2. ~0.8초 뒤 **양피지 브리프**가 펼쳐짐.
3. 펼침 시: **종이 펼치는 사운드** (`parchment-unfold`).  
   환영 편지 포함, 양피지가 열리는 모든 상황에 동일.
4. 접기 CTA: **종이 접는 사운드 + 날아가는 사운드** (`parchment-fold` → `parchment-flyaway`).
5. 본문은 **논픽션·공개 기록 기준** (주장/인정/관측을 구분, 사실 왜곡 없음).
6. 허브 nav는 우측 뉴스 Intel 사이드바를 **열지 않음** — 설명은 양피지가 담당.

**속보 타전음 (`hero-breaking`)** — 아래만 추가:
- 주장 · 영향 지역
- 반서방국 충돌사  

(긴급 속보 히어로와 같은 SOS 모스 전보음. 실시간 속보 도착을 뜻하지 않음.)

---

### 2-A. 「국경 · 우군 관계망」 클릭

1. soft fly → 허브 bbox.
2. 해당국(+우군) 폴리곤 강조 · 축 관계망 호(허브 스포크).
3. **양피지** — 국경·파트너 네트워크 논픽션 브리프.
4. 접으면 끝(지도 강조는 유지 가능). 패널 없음.

---

### 2-B. 「우군 국가」 항목 클릭

1. 우군 좌표로 fly · 허브↔우군 하이라이트.
2. **양피지** — 해당 우군과의 공개 관계 라벨·성격 설명.
3. 접으면 끝.

---

### 2-C. 「영유권 주장 및 영향」 클릭

1. claim bbox fly · **claim 펄스 링**.
2. **양피지** + **속보 타전음**.
3. 내용은 당사자 공식 주장 vs 국제적 미승인·중첩 주장을 분리 서술.
4. 접으면 끝(링은 유지 가능).

| 허브 | 대표 인식 | 번들 예 |
|------|-----------|---------|
| 중국 | 대만 | 대만, 남중국해, 센카쿠, 아루나찰 |
| 러시아 | 우크라이나 | 크림, 돈바스, 압하지야, 남오세티야 |
| 북한 | 남한/한반도 | 한반도 전체, 서해 NLL (2023 통일 포기 선언 반영) |
| 이란 | 호르무즈 | 걸프·호르무즈, 톤브, 레반트 |

---

### 2-D. 「무기거래 (SIPRI)」 클릭

1. 허브 fly · SIPRI 이전 호.
2. **양피지** — SIPRI·TIV·데이터 한계 설명.
3. 접은 뒤 **우측 리스트 패널**로 연도·장비 상세(선택적 심화).

---

### 2-E. 「반서방국 충돌사」 클릭

1. soft fly · 축 호 OFF.
2. **양피지** + **속보 타전음** — **상황 설명** 중심 브리프(진영 내부 충돌이 왜 중요한지, 11대 현장 개요).
3. 접은 뒤 **우측 패널**: 허브 렌즈 에피소드(+ 공통 체크).
4. 카드 클릭 → **현장 좌표** soft fly (pitch/bearing) → ~0.75초 뒤 **에피소드 양피지** + 펄스 링.
5. 11대: 전바오(1969)·랑선(1979)·갈완(2020)·츠힌발리(2008)·슈샤(2020)·샤트알아랍(1980–88)·톰브(1971)·프놈펜(1978)·바드메(1998–2000)·백두산 조중 긴장(1960s말)·두만강 북소 마찰(1980s).
6. V-Dem 전수 연표는 UI에서 사용하지 않음. 북한 2건은 공개자료 한계를 주석으로 명시.

## 3. 지정학 — 지구본 직접 조작

### 3-A. 탐색
1. 드래그 회전 · 스크롤 줌.
2. LOD에 따라 라벨·경로 밀도 변화.
3. 빈 바다 더블클릭 등으로 확대(구현된 제스처).

### 3-B. 전쟁·긴장 구역
1. 빨강/주황 빗금 박스에 hover → 이름·등급 툴팁.
2. 클릭 → 우측/Intel에 개요·당사자.
3. 가까이 줌 → 세분 세그먼트 우선.

### 3-C. GDELT 뉴스 핀
1. 전투(빨강) · 외교(주황) · 동맹(틸) · 시위 등.
2. hover: 뉴스 뱃지·제목.
3. 클릭: 해당 좌표 fly + Intel 시트(전장 필터).
4. S/A 중요도 강조 가능.

### 3-D. 레이어 패널 (≡)
- **체크 즉시 지도 반영** (패널을 닫을 때까지 기다리지 않음) — 지정학·지경학 공통
- 사용자가 수동으로 ON/OFF 예:
  - 동아시아 ADIZ
  - 축 관계망(전역 토글) — 허브 nav와 별개 레이어 스위치
  - NEPTUN / 우크라 전선 / FIRMS / 텔레그램 / 항모 / AIS · **항로·파이프·케이블·항구·크리티컬 노드** …
  - **「지나간 미사일·드론 궤적」체크박스 제거** — 강제 OFF
- **동시 ON 캡:** 일반 모드 무제한 · Ultra-Lite 12개

### 3-E. 전장 밀착 (프리셋·줌)
- 뷰포트가 우크라/대만/한반도/중동에 들어가면 **battlefield soft preset**이 레이어를 맞출 수 있음.
- 예: 중동 → AIS·항로·초크 쪽, 동아시아 → ADIZ 등.
- NEPTUN 궤적·체바 아돔 등은 해당 레이어·지역 조건에서만 체감.

### 3-F. 하단·속보
- GDELT 경보 패널 / 로컬 분쟁 경보.
- Telegram OSINT (ON 시).
- 범례: 전쟁·외교·동맹·시위 색.

### 3-G. Intel 전체 화면
1. 📰 / 속보 히어로 → 뉴스 시트.
2. 전장 칩으로 필터.
3. 기사 「지도」 → 시트 닫고 fly-to.

## 4. 지경학 — 주요 시나리오

### 4-A. 첫 화면
- 해운로·초크포인트·항구가 보이기 쉬움.
- 전선 UI·GDELT 전투 핀은 전면에 두지 않음.

### 4-B. Nav / 검색
1. 검색창 포커스 → 에너지·초크·허브 그룹 메뉴.
2. 호르무즈 / 수에즈 / 말라카 / 금융허브 등 선택 → **설명 모드**: soft fly(전망대 pitch≈55°, bearing≈−20°) + **양피지 스코어보드**(Risk / Impact / Market Link) + Critical Nodes 강조(선택·1-hop cascade).
3. 양피지 CTA: 접고 뉴스 패널 / 지도만 남기기. 양피지가 열린 동안 RSS 패널은 뒤로 둔다.
4. 검색 결과(지명·허브) 선택 → fly.

### 4-C. 하단 시장
- 티커(지수·VIX·유가).
- 경제 RSS / 관련 시장 패널.
- 「투자 힌트」성 문구는 교차확인 권고.

### 4-D. 레이어
- 파이프·LNG·AI DC·경제중심·**크리티컬 노드(MIT Atlas, 양 모드 공통)** 등.
- 지정학에서도 Critical Nodes 레이어를 켤 수 있음(클릭 시 축소 양피지).
- 지정학 전용 레이어는 강제 OFF되는 항목 있음.
- `STATSOFTHEWORLD_API_KEY`가 있으면 양피지에 국가 GDP·인구 보조 카드.

## 5. 모드 전환 (게이트 이후)

1. `ViewModeSwitcher` 등으로 **지정학 ↔ 지경학** 전환.
2. 크로미·fetch·하단 스택이 통째로 바뀜.
3. 허브 선택 상태·경제 nav 선택은 모드에 따라 리셋/분리.

## 6. 모바일·컴팩트

1. 좁은 화면 → compact UI.
2. 전선/경제용 **칩 프리셋**으로 레이어 단순화.
3. hover 툴팁 약화, 탭·클릭 중심.
4. **지정학 + 우크라 전선 포커스**여도 하단 📰 FAB은 유지(티커·범례만 접음). 데스크톱은 전선 중 하단 스택을 숨김.

## 7. 실패·저하 시나리오

| 상황 | 사용자 체감 |
|------|-------------|
| 로컬 데이터 로드 실패 | 상태 문구, 지구본은 뜰 수 있음 |
| GDELT 오류/지연 | 핀 부족, 로컬 분쟁 패널로 폴백 가능 |

| ChunkLoadError (배포 꼬임) | 새로고침·캐시 필요 (운영 이슈) |

## 8. “사이트 한 바퀴” 추천 데모 경로 (제품 스토리)

의도된 데모 순서:

1. 진입 → **지정학** 선택.  
2. 광역에서 전쟁/긴장 빗금·GDELT 점 확인.  
3. **중국** → 국경·우군.  
4. 주장 → **대만** 링.  
5. **무기거래** 패널·호.  
6. **반서방국 충돌사**에서 갈등 시점 카드 → soft fly.  
7. **러시아** → 크림/돈바스.  
8. 모드 전환 → **지경학** → 호르무즈·티커.  
9. 다시 지정학 → 레이어에서 ADIZ/NEPTUN 등 “깊게” 키기.

이것이 “멋진 신세계”의 **지정학 창 / 지경학 창** 이중 구조입니다.

## 9. 일부러 안 하는 것 (제품 경계)

## 10. 한 장 요약

```
[ 전원 공통 ]
방문 → 로딩 → 환영(양피지) → [지정학 | 지경학]
      → 광역 히어로 (전쟁·긴장·GDELT / 또는 항로·티커)
      → 표지① 상단 nav 설명 → 표지② 하단 뉴스 설명
      → 여기까지 끝나면 조작 가능

[ 유저가 nav를 직접 발견·클릭할 때만 ]
      ├─ 국경·우군 → fly → 양피지
      ├─ 우군 항목 → fly → 양피지
      ├─ 주장 링 → fly → 양피지 + 속보음
      ├─ SIPRI → fly → 양피지 → (접은 뒤 목록)
      └─ 충돌사 → fly → 상황설명 양피지 + 속보음 → 11대 카드 → 현장 fly + 양피지 + 펄스
```

로딩·게이트 중에는 허브/전장으로 자동 fly·양피지 금지.  
양피지 접기 CTA마다: 접는 소리 + 날아가는 소리.

사용자가 기억하는 문장:

> **지정학으로 축과 전선을 보고, 지경학으로 돈과 물류를 본다.**  
> **먼저 지도를 보고, nav와 하단 뉴스는 표지로 배운 뒤, 스스로 연다.**
